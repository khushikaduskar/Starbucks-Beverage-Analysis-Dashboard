# ☕ Starbucks Beverage Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** created to analyze Starbucks beverage data. It helps users understand beverage categories, nutrition values, caffeine levels, calories, sugar content, and product distribution through attractive visualizations.

The dashboard provides meaningful insights that can help in understanding beverage trends and comparing nutritional information.


## 🎯 Objectives

- Analyze Starbucks beverage nutritional information.
- Compare calories, sugar, and caffeine across beverages.
- Identify beverages with the highest caffeine.
- Visualize beverage category distribution.
- Build an attractive and interactive Power BI dashboard.

## 📂 Dataset Information

The dataset contains information such as:

- Beverage Name
- Beverage Category
- Calories
- Sugar (g)
- Protein (g)
- Caffeine (mg)
- Beverage Preparation Type


## 📈 Dashboard Features

### KPI Cards
- Total Beverages
- Average Sugar
- Average Calories
- Average Caffeine

### Charts Used

- Area Chart
- Bar Chart
- Donut Chart
- Horizontal Bar Chart
- Top 5 Beverage Cards
- Interactive Filters (Slicers)


## 📊 Key Insights

- Total beverages analyzed: **33**
- Average Calories: **194.30**
- Average Sugar: **33.02 g**
- Average Caffeine: **95 mg**
- Classic Espresso beverages contain the highest caffeine.
- Smoothies and Frappuccinos generally have higher calorie values.
- Beverage categories are evenly distributed across multiple product types.
